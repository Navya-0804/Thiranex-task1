# Sales & Revenue Analysis Dashboard

## Overview
This project analyzes sales and revenue data using Python.

## Features
- Import sales data from CSV
- Calculate KPIs:
  - Total Sales
  - Total Revenue
- Visualize monthly revenue trends
- Identify top-performing products

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python sales_dashboard.py

## Expected Outcome
Learn data visualization, KPI tracking, and business insight generation.
